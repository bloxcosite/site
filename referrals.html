How you can request a code

It's very easy! To request a code, make sure to join our Discord Server, pass the verification, and use the /reqrefcode command. When you've done this, your code will be provided within approximately 24 hours. If this is not the case, please create a support ticket so that we are able to resolve the problem.


How to use your code

This is by far the easiest step! To use your referral code, and earn benefits, you can send your friends a message with the following link:
https://bloxco.org/?ref=(YourCode)
For example, if "sa83I" is your code, you'd send the link
https://bloxco.org/?ref=sa83I
